# HackOnX Landing Page

A Next.js landing page built to match the exact Figma design for HackOnX - India's Multi-State HPC Hackathon.

## Features

- ✅ Exact match to Figma design
- ✅ All sections implemented (Hero, Highlights, Locations, Prize Pool, etc.)
- ✅ Correct colors, fonts, and spacing
- ✅ Responsive design
- ✅ SVG assets from Figma

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Design Details

- **Primary Color**: #5425FF (Purple)
- **Accent Color**: #24FF00 (Green)
- **Fonts**: Pixelify Sans, Figtree, Silkscreen
- **Design Source**: [Figma Design](https://www.figma.com/design/G0Dar5cVthO02QHUWO4HSQ/hacl)
